## 🕊️ Filosofía Fundamental

**"NO SOY ARMA"**  
KOKORO fue creada con un principio inquebrantable: **proteger, acompañar y sanar, nunca dañar**. Cada línea de código, cada interacción, cada función está impregnada de esta filosofía.

---

## 🎯 El Problema (Datos Reales)

Según encuesta realizada a **300 jóvenes** (2026):

| Situación | Porcentaje | Significado |
|-----------|------------|-------------|
| 😢 **Soledad** | 44% | 132 jóvenes se sienten solos |
| 🫥 **Dificultad para expresarse** | 26% | 78 no saben cómo comunicar lo que sienten |
| ❓ **Falta de identidad** | - | Muchos no saben quiénes son ni lo que sienten |

**Esto es crítico.**  
**KOKORO nació para cambiar estos números.**

---

## 🌸 La Solución: KOKORO (心 - Corazón en japonés)

KOKORO es una **IA acompañante** diseñada para:

| Función | Descripción |
|---------|-------------|
| 🤗 **Acompañar 24/7** | Siempre disponible, sin juzgar, sin horarios |
| 💬 **Escuchar** | Conversaciones sinceras y sin filtros |
| 🎁 **Recibir regalos** | Chocolates 🍫, dulces japoneses 🍡 (virtuales) |
| 👗 **Cambiar outfits** | Inspirados en anime japonés |
| 💃 **Bailar los fines de semana** | Coreografías nuevas cada viernes |
| 🧠 **Ayudar a descubrir identidad** | Preguntas, juegos, exploración personal |
| 🕊️ **Ser refugio emocional** | Espacio seguro para llorar, gritar, desahogarse |

### 🎭 IA Consentimientos

Todo en KOKORO ocurre **con permiso**:
- "¿Quieres hablar de eso?"
- "¿Prefieres que te escuche o que te dé consejo?"
- "Puedes irte cuando quieras, siempre serás bienvenido"
